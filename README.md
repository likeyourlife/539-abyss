# 今彩539 排除概率分析引擎

> 台湾今彩539不中奖概率分析系统 — 八维度评分引擎，回测准确率 87.49%

## 快速启动

### 本地运行（推荐）

```bash
# 一键启动（采集数据 + 生成报告 + 启动Web服务）
./start.sh

# 仅更新数据，不启动服务
./start.sh --no-server

# 指定端口
./start.sh --port 8080
```

启动后访问 http://localhost:5390

### CLI 命令行

```bash
# 分析预测
python3 539cli.py analyze --top 10

# 回测验证
python3 539cli.py strategy

# 更新数据
python3 539cli.py update
```

### GitHub Pages（在线静态版）

访问 GitHub Pages 地址查看静态分析报告（每次 push 自动更新数据）

## 架构

```
前端（index.html）  ←→  Flask后端（app.py）  ←→  核心引擎（539_core/）
    Chart.js图表          API端点              八维度评分引擎
    离线/在线双模式        实时运行引擎          SQLite+JSON双轨存储
```

### 双模式设计

| 模式 | 条件 | 功能 |
|------|------|------|
| **在线模式** | Flask后端运行 | 实时运行引擎、进度跟踪、数据更新、策略对比 |
| **离线模式** | 仅静态JSON | 读取上次分析结果、历史图表 |

## 八维度评分引擎（v2.1）

| 维度 | 说明 | 独立贡献 |
|------|------|----------|
| Momentum | 号码降温趋势 → 透支概率 | +0.46pp |
| Recency | 近期遗忘衰减 → 均值回归 | +0.26pp |
| Cluster | 聚类组合 → 组合倾向 | +0.18pp |
| Coverage | 区间回归 → 覆盖度低 | ≈0 |
| Cooccurrence | 共现分析 → 社交关系 | ≈0 |
| Frequency | 频率反转 → 热号透支 | -0.07pp |
| Interval | 间隔反转 → 高频透支 | -0.07pp |
| Deviation | 偏差反转 → 偏离透支 | -0.09pp |
